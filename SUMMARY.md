### 🔧 使用方法

#### 安装和运行
```bash
cd code-manager
npm install
npm run compile
# 按 F5 启动调试实例
```

#### 基本操作
1. **添加书签**: 在代码编辑器中右键 → "添加书签"
2. **创建集合**: 在文件资源管理器中右键 → "添加到集合"
3. **管理项目**: 点击左侧活动栏的"代码管理者"图标
