# Change Log

All notable changes to the "code-manager" extension will be documented in this file.

## [1.0.0] - 2024-02-11

### Added
- 🎉 初始版本发布
- 🔖 完整的书签管理功能（创建、编辑、删除、跳转）
- 📁 集合管理功能（文件分组、集合创建）
- 🎨 现代化的用户界面设计
- 🌈 自定义颜色和图标支持
- ⌨️ 右键菜单和键盘快捷键集成
- 💾 数据持久化存储
- 📱 响应式UI设计，适配VS Code主题

### Features
- **智能书签**：在代码任意位置添加带有颜色和图标的书签
- **精准跳转**：一键跳转到书签位置，支持跨文件导航
- **集合管理**：按功能模块组织相关文件到集合中
- **可视化界面**：直观的侧边栏面板和创建对话框
- **数据安全**：可靠的本地存储机制

### Technical
- 基于TypeScript开发，类型安全
- 使用Webpack构建，性能优化
- 遵循VS Code插件开发规范
- 模块化架构设计，易于维护扩展
